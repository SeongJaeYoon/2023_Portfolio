환상체 이상형 월드컵 웹페이지

환상체 이상형 월드컵 웹페이지는 Project Moon사의 Lobotomy Corporation 게임에 등장하는 캐릭터를 가지고 이상형 월드컵을 진행합니다. 첫 화면에서 16, 32, 64강을 선택할 수 있고, 선택 이 후에 캐릭터가 랜덤으로 정해지며 왼쪽 혹은 오른쪽의 마음에 드는 캐릭터를 선택해나가면 됩니다. 캐릭터를 모두 고르면 1위의 캐릭터가 보여지고 파란색 버튼을 누르면 첫 화면에서 선택한 n강을 다시 진행할 수 있으며 빨간색 버튼을 누르면 첫 화면으로 돌아갑니다.

개발 환경 : Visual Studio Code
